I. Общие правила
1. подготовка к запуску
a) запустить консоль nodejs
б) в консоли перейти в директорию приложения
в) набрать команду npm i

2. запуск приложения
а) запустить консоль nodejs
б) в консоле набрать node имя_файла.js [key если требуется]

II. Утилиты

1. Обнудение кошельков для биржи bittrex
a) настроить конфигурацилнный файл bittrex_conf.json в папке data
внеся соответсвующие данные

key = ключ API
sig = секретный ключ API

ВНИМАНИЕ! дайте ключу соответсвующие права для торговли

б) оновить список кошельков запустив утилиту node bittrex_curet.js
в) запустить утилиту для обнуления кошельков node bittrex.js

III. Аналитика по бирже Yobit
а) запуск node ./mod/analitic.js file_strategy запуск с именем файла стратегии, сбор данных
б) запуск node ./mod/analitic.js file_strategy stat - покажет число срабатываний минимального коэффицента за текущую дату
в) запуск node ./mod/analitic.js file_strategy stat 1.005 покажет число срабатываний с заданным коэффицента за текущую дату
г) запуск ./mod/analitic.js file_strategy stat 1.005 DD-MM-YYYY покажет число срабатываний с заданным коэффицента на указанную дату

д) формат файла стратегии для биржи Yobit
   "name": "Tesn" -- имя стратегии
  "lot": 0.005 -- минимальный лот
  "v_para": "eth_btc" -- валютная пара
  "period": "*/25 */1 * * * *" -- период проверки
  "power":  true -- стратегия включина, выключена - false
  "OrderBookLimit" :50 -- минимально число для просмотра ордеров
  "profit": 1.004 -- коээфицент соотношения для входа в рынок (сделка)
